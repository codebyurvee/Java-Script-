3. Swap Variables (3 Methods)

Method 1: Using Temp Variable

let a = 10, b = 20;
let temp = a;
a = b;
b = temp;

Method 2: Using Arithmetic

let a = 10, b = 20;
a = a + b;
b = a - b;
a = a - b;

Method 3: Using Destructuring

let a = 10, b = 20;
[a, b] = [b, a];
